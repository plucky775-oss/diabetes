# 당뇨관리 앱 프로토타입

Power TBM 스타일의 카드형 메인 화면을 참고해 만든 모바일용 당뇨관리 앱 프로토타입입니다.

## 포함 기능
- 홈 대시보드
- 공복혈당 / 식후 1시간 / 식후 2시간 기록
- 주간 / 월간 / 연간 그래프
- 식사 관리(거꾸로 식사법, 식이섬유, 당음료 체크)
- 운동 관리(식후 운동, 유산소, 근력)
- 혈액검사(HbA1c, 인슐린, 체중, 허리둘레)
- 생활습관(수면, 스트레스, 명상, 영양제 루틴)
- 리포트 요약
- 로컬 저장(localStorage)
- 간단한 PWA 구성(manifest, service worker)

## 실행 방법
1. 폴더를 압축 해제합니다.
2. `index.html`을 브라우저에서 열면 바로 확인할 수 있습니다.
3. PWA 설치/오프라인 캐시는 정적 서버에서 열 때 더 잘 동작합니다.
   - 예: GitHub Pages, Netlify, Vercel

## 참고
앱 구조는 업로드된 "당뇨 전단계 관리를 위한 로우데이터 테이블 명세"의 핵심 관리 범주를 바탕으로 구성했습니다.
